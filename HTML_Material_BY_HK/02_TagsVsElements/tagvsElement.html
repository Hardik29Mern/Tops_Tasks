Tag VS element

Tag

HTML Tags: Tags are the starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. Whatever written inside < and >are called tags.

HTML tags are used to hold the HTML element.

HTML tags are almost like keywords where every single tag has unique meaning.

Example :

<b> </b>

HTML Tags

tags are the keywords that are used to produce web pages in various formats. Opening tags and closing tags are found in the majority of tags.

  • HTML Headings :

    HTML headings are defined with the <h1> <h6> tags.

    <h1> defines the most important heading. <h6 > defines the least important heading

  • HTML Paragraphs :

    HTML paragraphs are defined with the <p> tag:

  • HTML Link :

    HTML links are defined with the <a> tag:

    The link's destination is specified in the href attribute.

  • HTML Images :

    HTML images are defined with the <img> tag.

    The source file (src), alternative text (alt), width, and height are provided as attributes.

HTML Elements

Elements enclose the contents in between the tags. They consist of some kind of structure or expression. It generally consists of a start tag, content and an end tag.

HTML element holds the content.

HTML elements specifies the general content.

Example :

<b> This is an element of html </b>

The HTML element is everything from the start tag to the end tag:

<agname>Content goes here...</tagname>

Examples of some HTML elements:

<h1>My First Heading</h1>

<p>My first paragraph.</p>
